# WriteWise AI

WriteWise AI is a simple AI chatbot project built for learning and practice.  
This project was created to explore how an AI-powered application works, starting from connecting an AI model, building a web interface, handling user input, and displaying chatbot responses in a clean way.

The main feature of this app is English grammar assistance. Users can type an English sentence or paragraph, and WriteWise AI will help correct grammar, improve wording, rewrite the text more naturally, and explain the changes.

This project is not intended to replace existing AI tools or professional writing assistants. It is mainly a practice project to better understand how to build and integrate AI features into a simple web application.

---

## Important Note

This project is only for learning and practice.  
It is not built to replace existing AI platforms, grammar tools, or professional writing services.

The goal is to understand the basic workflow of building an AI chatbot application using Python, Streamlit, and Gemini API.

The purpose of this project is to practice building a basic AI chatbot application.

Through this project, I learned how to:

- Connect a Python application with an AI model
- Use environment variables to store API keys safely
- Build an interactive UI with Streamlit

- Send user input to an AI model
- Display AI responses in a user-friendly format
- Improve the visual design of a simple web app
